% Acknowledgments
\section*{Acknowledgments}
We thank the Human Pangenome Reference Consortium (HPRC) for providing the high-quality haplotype-resolved assemblies that made this work possible. We acknowledge the developers of \texttt{impg}, minigraph-cactus, and AGC for the pangenome infrastructure underlying our approach.

% Data availability
\section*{Data Availability}
HPRC assemblies and pangenome graphs are available from \url{https://humanpangenome.org}. The HPRCv2-IBD software is available as open-source Rust code.

% Code availability
\section*{Code Availability}
Source code for all tools (ibd-cli, ibs-cli, ibs-from-paf, ancestry-cli, jacquard-cli) is available in the HPRCv2-IBD repository. The software includes 7{,}334 automated tests across 241 validation benchmarks.
