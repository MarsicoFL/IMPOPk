# Tutorial: `production/ibs-cli/scripts/jacquard_coeffs.sh`

## Purpose
`jacquard_coeffs.sh` reads IBS-positive windows and tallies the nine Jacquard
Delta states for two diploid samples (A and B). It answers questions such as
“what fraction of windows support Delta7 between these samples?”

## Prerequisites
- Sorted IBS TSV generated by `ibs.sh`/`run_full.sh` with columns
  `chrom start end group.a group.b estimated.identity`.
- GNU `sort` and `awk` (standard on most Unix systems).

## Required inputs
| Flag | Description |
| --- | --- |
| `--ibs` | Path to the IBS window TSV |
| `--hap-a1/--hap-a2` | Haplotypes for diploid sample A |
| `--hap-b1/--hap-b2` | Haplotypes for diploid sample B |

Haplotype names must match the identifiers used in the IBS file, typically of
the form `sample#hapIndex` or `sample#hapIndex#contig:coords`.

## How to run
```bash
cd production/ibs-cli/scripts
./jacquard_coeffs.sh \
  --ibs ../ibs_for_ibd.out \
  --hap-a1 HG01167#1 --hap-a2 HG01167#2 \
  --hap-b1 NA19682#1 --hap-b2 NA19682#2
```

## Output
- `stdout`: nine lines `Delta1..Delta9` with the fractional counts and raw
  window totals.
- `stderr`: metadata describing chromosome, coordinate span, theoretical window
  count, and how many windows fell into Delta9 (no IBS among the four haplotypes).

Use shell redirection to save the table: `./jacquard_coeffs.sh ... > jacquard.tsv`.
